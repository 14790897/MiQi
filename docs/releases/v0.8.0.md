# v0.8.0 版本更新说明

> 发布日期：2026-07-24

v0.8.0 是 0.7.0 之后的一次主要版本更新，涵盖了 7 月 15 日至 24 日期间从 develop 分支合并的大量修复和功能增强。

---

## ✨ 新功能

### 文档上传与解析
- **PDF / Office / Markdown 文档上传**：支持将文档上传到对话中，自动解析内容供 AI 分析，并可在系统中打开预览（[#372](https://github.com/14790897/MiQi/pull/372)）

### 任务管理
- **右键批量操作**：侧边栏"全部"标签页中支持右键批量删除/归档任务（[#430](https://github.com/14790897/MiQi/pull/430)）

### 桌面端
- **macOS 打包**：新增 macOS 平台支持，CI 自动构建 `.dmg` 安装包（[#386](https://github.com/14790897/MiQi/pull/386)）
- **内联终端输出**：设置中新增开关，可在对话中直接查看命令执行的终端输出（[#363](https://github.com/14790897/MiQi/pull/363)）

### 用户反馈
- **应用内反馈提交**：支持在应用中直接提交反馈到飞书多维表格，自动收集日志（[#321](https://github.com/14790897/MiQi/pull/321)、[#380](https://github.com/14790897/MiQi/pull/380)）

---

## 🐛 修复

### 核心运行时
- **Plan 模式规范化**：明确 Plan 模式为只读分析模式，完善能力测试，补充授权绕过安全说明（[#388](https://github.com/14790897/MiQi/pull/388)）
- **Agent 首 Token 超时**：防止 Agent 对话在流式输出开始前无限挂起（[#391](https://github.com/14790897/MiQi/pull/391)）
- **事件队列容量限制**：添加 `maxsize=4096` 防止事件队列无限增长（[#354](https://github.com/14790897/MiQi/pull/354)）
- **历史消息压缩顺序**：修复上下文压缩后消息顺序错乱的问题（[#158](https://github.com/14790897/MiQi/pull/158)）
- **Token 估算修正**：调整 estimator 估算比例，添加发送前 context guard（[#291](https://github.com/14790897/MiQi/issues/291)）
- **Turn 修剪机制**：修复 `trim_for_model` 时完整 turn 被破坏的问题（[#383](https://github.com/14790897/MiQi/pull/383)）
- **订阅泄漏**：修复 `subscribe()` 调用时序导致的订阅泄漏（[#352](https://github.com/14790897/MiQi/pull/352)）
- **会话删除清理**：删除会话时自动清理 `AppServer._subscriptions`（[#353](https://github.com/14790897/MiQi/pull/353)）
- **最大工具迭代次数**：从 100 提升至 500，适配长时间工作负载（[#318](https://github.com/14790897/MiQi/pull/318)）

### 桌面端
- **文档注入展示**：聊天历史中文档注入的原始文本改为彩色标签展示，不再直接暴露（[#426](https://github.com/14790897/MiQi/pull/426)）
- **流式请求超时**：为 `chat.send` 流式请求添加 inactivity 超时（[#319](https://github.com/14790897/MiQi/pull/319)）
- **事件生命周期过滤**：过滤 `chat.send` 泄出循环中的 exec 生命周期事件（[#325](https://github.com/14790897/MiQi/pull/325)）
- **终端事件处理**：确保 `onEvent` 始终执行 resolve（[#355](https://github.com/14790897/MiQi/pull/355)）
- **侧边栏会话列表**：移除硬编码的 20 个会话限制（[#368](https://github.com/14790897/MiQi/pull/368)）
- **文件打开方式**：改为系统默认应用打开文件（[#339](https://github.com/14790897/MiQi/pull/339)）
- **沙箱 IP 超时**：延长沙箱初始化方法 IPC 超时（[#311](https://github.com/14790897/MiQi/pull/311)）
- **Session 方法超时**：防止沙箱初始化期间 session 方法超时（[#272](https://github.com/14790897/MiQi/pull/272)）
- **设置清理**：修复清空设置字段时不生效的问题
- **轮询竞态**：为 plugins.list 轮询添加 in-flight guard

### 沙箱
- **日志增强**：为沙箱命令失败路径添加详细日志记录（[#410](https://github.com/14790897/MiQi/pull/410)）
- **Windows 控制台闪现**：修复 WSL 命令执行时控制台窗口闪现问题（[#314](https://github.com/14790897/MiQi/pull/314)）

### 国际化
- **全量中文化**：所有面向用户的错误消息统一翻译为中文（[#393](https://github.com/14790897/MiQi/pull/393)）

### 新手引导
- **API Key 缺失提示**：首次安装无 API key 时不再显示"内部错误"，改为引导用户配置模型（[#394](https://github.com/14790897/MiQi/pull/394)）
- **Hot Reload 改为手动**：桥接热重载改为 opt-in，避免开发时的自动重载干扰（[#240](https://github.com/14790897/MiQi/pull/240)）

### Bridge
- **SIGTERM 安全退出**：关闭前 await 所有资源销毁再清理状态文件（[#358](https://github.com/14790897/MiQi/pull/358)）
- **stderr 日志防护**：bridge 关闭时防止向已关闭 stderr 写入导致异常（[#359](https://github.com/14790897/MiQi/pull/359)）

### CI / 基础设施
- **Workflow 令牌修复**：使用 PAT 绕过 main 分支保护规则，确保 semantic-release 可正常推送版本文件（[#441](https://github.com/14790897/MiQi/pull/441)）
- **Issue 验证器**：移除字符数下限限制
- **全链路日志**：修复日志分类、ANSI 剥离、时间戳等细节（[#398](https://github.com/14790897/MiQi/pull/398)）
